# Examples

Real-world examples demonstrating Firehose patterns and capabilities.

## Available Examples

### [HTTP Server](/examples/http-server)

Build an HTTP event processing pipeline with request routing, middleware, and response handling.

**Topics:**
- HTTP event sources
- Request transformation
- Response destinations
- Error handling

### [Message Queue](/examples/message-queue)

Process messages from Kafka/RabbitMQ with filtering, transformation, and batching.

**Topics:**
- Message queue sources
- Event filtering
- Batch processing
- Dead letter queues

### [System Monitor](/examples/system-monitor)

Monitor system processes and send alerts based on conditions.

**Topics:**
- Process monitoring sources
- Conditional alerting
- Multiple destinations

### [Event-Driven Microservice](/examples/microservice)

Complete microservice architecture using event-driven patterns.

**Topics:**
- Multiple event sources
- Service composition
- Middleware chains
- Production deployment
